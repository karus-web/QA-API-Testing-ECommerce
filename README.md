# API Testing Project – End-to-End E-commerce Workflow

This project demonstrates API testing of an e-commerce application using Postman.

---

## 🚀 Project Overview
End-to-end testing of:
- Signup  
- Login  
- Order Placement  
- Payment Processing  

---

## 🧪 What I Tested
- API response validation  
- Authentication (token-based)  
- Transaction workflows  
- Data integrity across requests  

---

## 🛠️ Tools Used
- Postman  
- REST APIs  
- JSON  
- JavaScript (Postman test scripts)  

---

## 📌 Key Learning
- Chaining API requests using global variables  
- Testing secure endpoints  
- Validating real-world user flows  

---

## 🔗 API Workflow Covered

1. Customer Signup → Generates customer ID  
2. Login → Generates authentication token  
3. Order Placement → Uses customer ID + token  
4. Order Details → Validates order creation  
5. Payment → Processes transaction  
6. Cancel Payment → Validates cancellation  

This project demonstrates chaining of API requests and validation of transaction workflows.
